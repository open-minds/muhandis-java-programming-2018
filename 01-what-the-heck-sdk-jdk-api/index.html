To run Java you need JRE. And for developement you need a SDK. The Java's SDK is known as JDK. It contains the runtime environement and Java's compiler with a set of APIs and libraries , such as Swing, AWT or JavaFx GUI libraries...
You only need to download the right version of JDK from Oracle's website for your operating system.

What the Heck is ?

Java is cross platform thanks to the JRE

The Java runtime environment consists of the Java virtual machine, the Java core classes, and supporting files.

Java's SDK is called JDK

Compiler translate your code to a machine code

Library is a collection of programs

Example:
Calculate the square-root (جذر تربيعي) 

						
public class MathDemo {
   public static void main(String[] args) {
      double x = 9;

      // print the square root of x
      System.out.println("The square root is " + Math.sqrt(x));
   }
}

API is an interface

You don't know how Math.sqrt() is implemented, you just call the function and you use it.

Swing, JavaFX, AWT... Are libraries for making GUI

You won't program a button, you just need to use it in you projects. And Java libraries are here for you.

CLI != GUI

CLI is the console, the boring area where you interact with your programs and GUI are the interfaces where humans can interact with programs and computers.

IDE

Integrated development environment =
Source Code editor + Compiler + Debugger + plugins

It provides facilities such as smart autocompletion, importing libraries ...

Getting ready for the Java journey

Install JDK from Oracle's website

Choose an IDE (Intellij Idea, Eclipse, Netbeans)
